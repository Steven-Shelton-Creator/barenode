## Surprises / Learnings

- **The phase doc underestimated scope** — it said only `agent.py` would change, but the provider seam needed a full signature change across 4 implementations.
- **The real work was in `provider.py`, not `agent.py`.** The Agent change was truly 3 lines (`self.messages = []`, two appends). Updating all 4 provider backends took the bulk of the effort.
- **No REPL changes needed.** All history management is internal to the Agent.
- **Context tracking is fragile** — on session restart, the assistant lost track of where verification logs and workflow docs lived. This session's second pass was entirely fixing context drift.
