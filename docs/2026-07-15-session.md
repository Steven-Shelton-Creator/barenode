- `make_event()` — creates events with GenAI semantic conventions
- Event builders: `llm_call_event()`, `tool_call_event()`, `completion_event()`

### `src/harness/tracer.py`
- `Span` dataclass — name, start/end time, attributes, events, parent_id, span_id
- `Tracer` class — span lifecycle, `span()` context manager, `get_trace()`, `clear()`
- `ConsoleSink` — prints formatted trace after each turn (cost, tokens, duration)
- `JsonlSink` — writes trace to JSONL file
- `MultiSink` — fans out to multiple sinks

### `src/harness/agent.py`
- Wired tracer into `send()` — wraps model call and tool calls in spans
- Tracks token counts (estimated) and cost per call
- Prints trace to console after each turn via ConsoleSink

### `src/ui/app.py`
- `BarenodeApp` — Textual App with two-pane layout, input bar, header/footer
- `AgentRunner` — runs agent in worker thread, captures conversation + trace
- Uses `@work(exclusive=True, thread=True)` for non-blocking agent execution
- Uses `call_from_thread()` for safe UI updates from worker thread
