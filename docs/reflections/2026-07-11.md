## Surprises / Learnings

- **Build system trap:** Using `project.scripts` entry points requires the package to be built first. Had to add `[build-system]` with setuptools and `tool.setuptools.packages.find.where = ["src"]`.
- **Imports drop the prefix:** When packages are installed from `src/`, import paths become `harness.agent` not `src.harness.agent`.
- **Fake provider is invaluable:** Tests run in milliseconds with no internet or model needed.
- **The workflow cycle was identified organically** — not planned in advance, but observed after the fact.
